# 题目技巧

  1. 斐波那契。注意当 n 达到一定程度的时候，会超出安全数字范围，所以可能需要取余，取余的方式是 sum = (a % p + b % p) % p

  2. 取中位数的时候, 可以使用 Math.floor(lo + (hi - lo) / 2), 也可以直接使用 (lo + hi) >>> 1, 后者更简洁
